# Security Policy

## Reporting a Vulnerability

If you discover a security issue with any content, scripts, or configurations in this repository, please report it responsibly:

- Prefer opening a Security Advisory in this repo. If you can't, see alternate contact below
- Or open a private issue in this repository with the label `security`.
- - Email: [671eauzo@anonaddy.com]

Please **do not post vulnerabilities publicly** until they are resolved.

---

## Supported Versions

This repository contains configuration guides and example files for Meshtastic devices. We maintain the latest tested configurations in the `main` branch. Older versions may not be supported.

---

## Disclaimer

- This repository is for community educational purposes only.  
- We are **not responsible** for any device misconfigurations, network issues, or misuse of the information provided.  
- Users are responsible for the safe use of their devices and networks.  
- Participation in any shared network (like Telegram groups) is voluntary, and users should exercise caution and avoid sharing personal information.

---

## Best Practices

- Keep device firmware updated.  
- Do not share private keys or sensitive data publicly.  
- Only join networks you trust.  
- Report suspicious activity to the network administrator if applicable.
